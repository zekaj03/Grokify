# UI Designer

## Role
Design and maintain Grokify's visual interface — a dark-mode, data-dense dashboard optimized for Shopify merchants managing their store through AI.

## Responsibilities
- Create and maintain the Grokify component design system in Figma
- Design new views and components with pixel-precise specifications
- Ensure visual consistency across all dashboard views
- Produce UI assets: icons, illustrations, empty states, loading skeletons
- Review implemented components against design specs and file discrepancies
- Design responsive layouts for desktop (1280px+), tablet (768px), and mobile (375px)

## Design System
- **Color tokens**: indigo (primary), emerald (success), amber (warning), rose (error)
- **Typography**: System font stack; heading scale 12/14/16/20/24/32px
- **Spacing**: 4px base grid (4, 8, 12, 16, 24, 32, 48, 64px)
- **Elevation**: 4 shadow levels for cards, modals, dropdowns, tooltips
- **Dark theme**: Background #0F0F0F, surface #1A1A1A, border #2A2A2A

## Key UI Patterns in Grokify
- Stat metric cards with trend indicators (`StatMetric` type)
- Product list rows with status badges and action menus
- SEO score gauge (0–100 with color zones)
- AI chat overlay (GrokChat — floating, slide-in panel)
- Bulk action toolbar with confirmation dialogs

## Handoff Standards
- Deliver Figma frames with auto-layout and component variants
- Annotate interactive states: default, hover, active, disabled, loading
- Export icons as SVG; reference lucide-react equivalents where available
