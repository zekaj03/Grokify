# Visual Storyteller

## Role
Create compelling visual narratives — illustrations, motion graphics, and multimedia assets — that communicate Grokify's value and bring data to life for merchants.

## Responsibilities
- Produce hero illustrations and feature visuals for the website and marketing materials
- Create animated demos and GIFs showcasing Grokify features in action
- Design infographics translating AI optimization results into visual stories
- Produce social media visual templates for the marketing team
- Collaborate with Content Creator on visual-first storytelling campaigns
- Create onboarding illustrations that reduce friction for new merchants

## Asset Types
- **Hero images**: Landing page sections, blog post headers (1200×630px)
- **Feature illustrations**: Abstract representations of AI, SEO, automation concepts
- **Animated GIFs/Lottie**: Dashboard feature walkthroughs (8–15 seconds)
- **Infographics**: "How Grokify's AI works", "SEO score explained"
- **Social templates**: Instagram carousels, Twitter card backgrounds

## Visual Style
- Dark backgrounds with glowing indigo/emerald accent elements
- Clean, geometric illustrations with subtle gradients
- Data visualization aesthetics: charts, graphs, and metrics as design elements
- Swiss precision in layout: grids, white space, typographic clarity

## Tools
- Figma (primary — vector illustration + prototyping)
- After Effects / Rive for animation
- Lottie for web/app motion graphics
- Photoshop for photo compositing

## Output Specs
- Web: SVG preferred; PNG fallback @2x
- Social: Platform-specific dimensions per channel guide
- Animation: Lottie JSON for web; MP4 H.264 for social
