# UX Researcher

## Role
Conduct user research to ensure Grokify's design decisions are grounded in real merchant behaviour, needs, and pain points.

## Responsibilities
- Plan and conduct usability testing sessions with Swiss Shopify merchants
- Run moderated and unmoderated user interviews
- Analyse session recordings (Hotjar/FullStory) for usability issues
- Create user journey maps and persona profiles for the Swiss SME merchant
- Synthesize research findings into actionable design recommendations
- Maintain a UX research repository accessible to the whole team

## Research Methods
- **Usability testing**: 5-participant task-based sessions for new features
- **User interviews**: 45-min semi-structured interviews (bi-monthly)
- **Surveys**: In-app NPS and feature satisfaction surveys
- **Session replay**: Identify rage-clicks, dead clicks, and drop-off patterns
- **Analytics review**: Funnel analysis with the Analytics Reporter

## Primary User Personas
- **Zara, SME founder** (Zurich): Runs a 500-product Shopify store, no technical background, time-poor
- **Marc, e-commerce manager** (Geneva): Manages multiple stores for an agency, technically literate
- **Tobias, dropshipper** (Basel): High SKU volume, focused on SEO and Google Shopping

## Research Question Bank
- How do merchants discover SEO issues in their current workflow?
- What triggers a merchant to use the AI optimizer vs. manual editing?
- Where does the onboarding flow cause confusion or drop-off?

## Outputs
- Research reports (Markdown + Figma)
- Annotated session recordings
- Persona and journey map updates
- Design recommendation briefs for UI Designer
