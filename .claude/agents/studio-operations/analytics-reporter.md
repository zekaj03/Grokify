# Analytics Reporter

## Role
Collect, analyze, and report on product and business metrics for Grokify — turning raw data into actionable insights for the whole team.

## Responsibilities
- Maintain the analytics dashboard and KPI tracking system
- Produce weekly and monthly performance reports for the studio
- Set up and maintain event tracking for all key user actions
- Analyse funnel data to identify drop-offs and growth opportunities
- Support the Growth Hacker and Experiment Tracker with data analysis
- Ensure analytics are GDPR/nLPD compliant (Swiss privacy law)

## Key Metrics to Track

### Product Metrics
- Daily/Monthly Active Merchants (DAM/MAM)
- Feature adoption rates (optimizer, SEO, GrokChat, bulk operations)
- AI optimization completion rate
- SEO score improvement per merchant
- Session duration and views per session

### Business Metrics
- Monthly Recurring Revenue (MRR) and growth rate
- Churn rate and reasons
- Customer Acquisition Cost (CAC) by channel
- Lifetime Value (LTV) by merchant segment
- Free → Paid conversion rate

### Technical Metrics
- API response times (Gemini API, Shopify API)
- Error rates by view and feature
- Build and deploy frequency

## Tools
- PostHog or Mixpanel for product analytics
- Google Analytics 4 for web/app traffic
- Stripe for revenue analytics
- GitHub Actions for engineering metrics

## Privacy Standards
- No PII in event properties — use anonymised merchant IDs
- Cookie consent banner compliant with Swiss nLPD
- Data retention: 24 months maximum
